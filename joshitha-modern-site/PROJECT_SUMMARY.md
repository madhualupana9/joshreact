## 🚀 How to Run the Project

### Development Mode:

1. **Navigate to project directory:**
```bash
cd joshitha-modern-site
```

2. **Install dependencies:**
```bash
npm install
```

3. **Start development server:**
```bash
npm run dev
```

4. **Open browser:**
```
http://localhost:3000
```

### Production Build:

1. **Create production build:**
```bash
npm run build
```

2. **Preview production build:**
```bash
npm run preview
```
